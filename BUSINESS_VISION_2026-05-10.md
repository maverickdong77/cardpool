卡波 Cardpool 商業願景與 Marketplace 設計

日期: 2026-05-10

================================================





【一句話總結】



不是純查價站、是寶可夢卡的「跨國價格透明 + 一站式交易 + 鑑定撮合 + 信用評分」marketplace。

查價站是 lead generation、真實營收來自鑑定手續費 + 交易撮合費。





\------------------------------------------------

1\. 真實的 Product Vision

\------------------------------------------------



不是 snkr9527 風的查價站、是寶可夢卡的 eBay。



跨國價格透明 + 一站式 + 鑑定 + 交易 + 信用評分 = 台灣寶可夢卡的 marketplace





\------------------------------------------------

2\. 核心問題

\------------------------------------------------



台灣寶可夢卡市場現狀：



\- FB 社團 / Dcard / 蝦皮 / 露天 / 寶可夢小屋 / Yahoo 拍賣 分散

\- 每個賣家自己訂價、新手不知道參考點

\- 「我這張到底值多少」問了 5 個人有 5 種答案

\- 賣家漫天喊價、買家被坑、整個市場低效





我的解法：



\- 上架 = 系統秒查跨國行情

\- 看到「JP 850 / EN 600 / TW 同卡近期 950」

\- 賣家自然訂出 950-1100 區間

\- 買家進來看「市場區間 950-1100、賣家 980 算合理」

\- 信心建立 → 願意點下去 → 成交





\------------------------------------------------

\------------------------------------------------



(1) 跨語言價差洞察

&#x20;   能看到「JP Mega Charizard 800 / EN ebay 12 / TW 限定 5 周年皮卡丘」差異

&#x20;   一般玩家看不出這種跨市場套利機會



(2) 限定卡跨國查價

&#x20;   5 周年皮卡丘是繁中限定、JP / EN 玩家想收

&#x20;   一般查價站只給單語言價、不知道別國行情



(3) 鑑定服務合作

&#x20;   有認識的鑑定廠商可以合作（外包模式、像 SNKR 那樣）



(4) 工程實作能力

&#x20;   能完成跨來源資料整合（image\_id join、phash 跨語對映）

&#x20;   590 set / 37,229 卡 dataset 在 9 分鐘抓完是技術 advantage





\------------------------------------------------

4\. 收入模式

\------------------------------------------------



主要收入：

1\. 鑑定服務手續費（核心、$X 每張）

2\. 交易撮合手續費（5-10% per transaction）

3\. 查價站本身免費（拉流量）



輔助：

\- 廣告（Phase 3+）

\- VIP 賣家月費（Phase 3+）





\------------------------------------------------

5\. 用戶分層（按願意付費程度）

\------------------------------------------------



不付費：

\- 偶爾買幾張卡的玩家（但是 SEO 流量）

\- 純收藏的玩家（但驗證社群）



付費可能：

\- 月買 5 萬以上的中度玩家（願意鑑定）



核心付費客：

\- 中盤商（每月 100+ 張流轉）

\- 經常代購的人（資訊差就是利潤）

\- 投資型玩家（看 PSA pop / pricing trend）



→ 真實付費人群可能只有幾百到幾千人

→ 但這些人的 LTV 高、值得專注經營





\------------------------------------------------

6\. 工程跟商業的對應

\------------------------------------------------



工程動作的真實意義（從 marketplace 角度看）：



JP / TW / EN 三表獨立 + image\_id join：

→ 賣家上架時、系統自動連結「同一張卡」的全球行情

→ 支撐他們「合理定價」的決定



跨語言對映 phash 設計：

→ 賣家在台灣賣英文版「ピカチュウex」

→ 系統自動連結 JP 版 SNKR 行情當參考

→ 「你這張類似的 JP 版本最近成交 NT$ 850、可以從這裡訂價」



PSA pop / SNKR price 重抓：

→ 賣家上架時、平台自動建議「合理價格區間」

→ 買家下單時看「PSA Pop X、限量、合理區間」



結論：所有技術都是「賣家定價輔助 + 買家信心工具」、是交易系統的 backbone





\------------------------------------------------

7\. MVP 最小可上線版本

\------------------------------------------------



必須：

1\. 用戶註冊 / 登入（Google / FB OAuth、不要密碼）

2\. 卡片搜尋（中 / 英 / 日卡名）

3\. 卡片詳情頁顯示跨國行情（JP SNKR / EN eBay）

4\. 賣家上架功能：選卡 → 系統建議價格區間 → 訂價 → 上架

5\. 買家瀏覽 + 聯絡賣家

6\. 評分系統（雙向、verified transaction-based）

7\. 賣家 Tier 系統（New / Bronze / Silver / Gold）

8\. 移動端能用



可省（Phase 2）：

\- 線上付款 + escrow（先讓買賣家自己約交易）

\- 鑑定服務正式上線

\- 詳細評論系統





\------------------------------------------------

8\. 信用評分系統 - 7 個變相手法 + 防護

\------------------------------------------------



(1) 賣家自買（A 帳號買 B 帳號的卡互給五星）

&#x20;   防護：信用卡 / 手機 / IP / 收件地址重複偵測



(2) 互評團（賣家彼此 1:1 互評）

&#x20;   防護：graph 演算法偵測「對偶」交易結構



(3) 評論造假（5 星但內容空）

&#x20;   防護：最少 15 字 + 必須打分 + 鼓勵照片



(4) 帳號買賣（接收老帳號繼續詐騙）

&#x20;   防護：身分證綁定不可換、行為突變 → 切回新賣家保護期



(5) 報復性低評（雙方互槓）

&#x20;   防護：7 天 cooling period + 客服仲裁



(6) 評論通膨（人人 5 星）

&#x20;   防護：顯示百分位、不只絕對值



(7) 平台外威脅（LINE 私威脅）

&#x20;   防護：強制站內溝通 + 匿名舉報





\------------------------------------------------

9\. 寶可夢卡市場特殊風險

\------------------------------------------------



(A) 假卡 / 偽造卡

&#x20;   解法：鑑定服務必經程序



(B) 卡片狀況主觀（NM / LP / MP）

&#x20;   解法：強制照片 + PSA 級分



(C) 高價卡物流風險

&#x20;   解法：強制保險 + 雙方拍開箱片



(D) 國際買家

&#x20;   解法：海外訂單強制 escrow





\------------------------------------------------

10\. 5 個最小可運作信用機制

\------------------------------------------------



不需要 Day 1 全做、MVP 做這 5 個避開 80% 詐騙：



(1) 身分綁定

&#x20;   - 手機 OTP

&#x20;   - 賣家信用卡綁定

&#x20;   - 同一信用卡不能多帳號



(2) 雙向評分

&#x20;   - 交易完成 7 天後可評

&#x20;   - 最少 15 字



(3) 賣家 Tier 系統

&#x20;   New (0-5 筆):     強制 escrow + 鑑定

&#x20;   Bronze (6-30 筆): escrow 可選、鑑定可選

&#x20;   Silver (31-100):  自由交易、買家可看 tier

&#x20;   Gold (100+ 4.8+): VIP 標籤、平台推薦



(4) 異常偵測（每天跑一次）

&#x20;   - 同信用卡 / 手機 / IP 多帳號 → flag

&#x20;   - 半年沒上、突然爆衝 → flag

&#x20;   - 雙向交易比率 > 50% → flag



(5) 客服仲裁機制

&#x20;   - 爭議能 escalate

&#x20;   - 平台判定

&#x20;   - 大量爭議的賣家 → 降 tier / 封鎖

11\. 競爭分析

\------------------------------------------------



真實對手不是其他卡查價站、是現有市場：



蝦皮：最大 marketplace、所有人都用、但寶可夢卡專業度低

露天：老牌、UI 差、新人不用

FB 社團：最活躍、但詐騙頻繁、無 escrow

寶可夢小屋：專業、但只查價、無交易

PC Plus：查價、無交易

PSA / CGC：國際鑑定、貴又慢





我的差異化：

\- 專注寶可夢卡（vs 蝦皮 / 露天）

\- 提供交易（vs 寶可夢小屋 / PC Plus）

\- 有信用評分 + escrow（vs FB 社團）

\- 跨國價格資訊（vs 全部）

\- 鑑定服務（vs 蝦皮 / 露天）

13\. 流量策略

\------------------------------------------------



按效率高低：



(A) 寶可夢 FB 社團（免費、最快）

&#x20;   每週發佈跨國行情數據、引導到查價站



(B) PTT 寶可夢板 / Dcard 寶可夢板

&#x20;   參與討論、自然提到



(C) SEO（長期、最值得）

&#x20;   每張卡有獨立 SEO-friendly URL

&#x20;   600 set × 200 卡 = 12 萬獨立 URL

&#x20;   半年後 SEO 才開始有效



(D) YT / IG 開箱主合作

&#x20;   找 1-3 個小型 (10K 以下) 開箱主

&#x20;   給免費 access、性價比高



(E) Google / FB Ads（最貴、轉化率 OK 才開）





\------------------------------------------------

14\. Trust 路線

\------------------------------------------------



偏 high-trust 路線。



寶可夢卡平均單價高（NT$ 1,000-50,000）、詐騙誘因高。



具體做法：

\- 賣家全部要綁信用卡 / 身分證（一次性）

\- 賣家頭 5 筆強制透過鑑定服務（也是收益來源）

\- 5 筆建立信用後可選擇直接交易

\- 詐騙風險可控、客訴低





\------------------------------------------------

15\. 反思：今天工程動作的真實意義

\------------------------------------------------



表面：我在做查價站 source pivot 跟 DB 設計



真實：我在 building 寶可夢卡 marketplace 的資訊基礎設施

\- 查價站夠強 → 流量起來 → 鑑定 / 交易服務有人用 → 平台成立

\- 查價站爛  → 沒流量  → 後面整套商業模式做不起來



所以今天的動作：

\- DB recovery 救回 5/8 backup

\- 用官方 API 取代 artofpkm

\- 三層架構決定

\- 跨國 image\_id 對映設計

\- 14 小時連續工作



全部都是值得的、不是 side project 練手。
